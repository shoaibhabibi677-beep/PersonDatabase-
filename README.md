# PersonDatabase-
این پروژه یک سیستم ساده و قابل توسعه برای ثبت، ذخیره و مدیریت اطلاعات اشخاص می‌باشد. هدف از این سیستم، ایجاد یک دیتابیس منظم است که بتوان اطلاعات افراد را به‌صورت دقیق ثبت، جستجو و ویرایش کرد. فایل فشرده را از قسمت ریلز دانلود کرده و استتفاده کنید برنامه را با اسکریپت run pp.bat اجرا کنید تمام just clik on run app.bat

دیتابیس ایجاد و مدریریت افراد

امکانات این نسخه
امکانات ایجاد فرم:
شامل آپلود عکس، معلومات هویتی، معلومات تماس و آدرس، معلومات تحصیلی، ایجاد بخش دلخواه همراه با فیلد هایش، آپلود عکس و PDF، ایجاد یادداشت های حرفه ای و با امکانات زیاد.
قابلیت جستجو پیشرفته با نام، تگ ها، شماره تذکره ...
قابلیت فیلتر اطلاعات افراد بر اساس جنسیت، ولایت، دسته بندی ها، و تگ ها
قابلیت نمایش و فیلتر یادداشت ها و جستجو آنها
قابلیت آپلود اسناد مثل عکس، و PDF
قابلیت های که در تنظیمات گنجانیده شده: شامل انتخاب زبان فارسی/انگلیسی، آپلود یا خروجی گرفتن از دیتابیس، خروجی گرفتن از دیتابیس به شکل راپور با فرمت html ، معلومات توسعه دهنده و تغیر رمز عبور
